# Peng

**Pong game clone in raylib**
This is my first project using raylib, I did it to test and learn about the lib

![](https://raw.githubusercontent.com/Teb0so/peng/refs/heads/main/art/peng.gif)

# Building

1. Build raylib
```
cd peng/raylib/src/
make PLATFORM=PLATFORM_DESKTOP
```

2. Build the game
```
cd peng
make
```
